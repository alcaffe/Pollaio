È un’app GUI completa con:

campi per Data (preimpostata ad oggi) e Numero uova
pulsanti Salva, Aggiorna, Elimina, Pulisci
tabella con l’elenco delle registrazioni
creazione automatica del database/tabella se non esiste
validazioni di base (data in formato YYYY-MM-DD, numero intero ≥ 0)
Python 3.x
Moduli standard (nessuna installazione extra): sqlite3, tkinter, datetime, csv
